Quick Installation

Get up and running in minutes with these simple commands

Ubuntu/Debian 
curl -s https://packages.fluentbit.io/fluentbit.key | apt-key add -
apt-get update
apt-get install fluent-bit
CentOS/RHEL 
curl -s https://packages.fluentbit.io/fluentbit.repo -o /etc/yum.repos.d/fluentbit.repo
yum install fluent-bit
Homebrew 
brew install fluent-bit
Docker Run 
docker run -d --name fluent-bit fluent/fluent-bit:latest
Helm Installation 
helm repo add fluent https://fluent.github.io/helm-charts
helm install fluent-bit fluent/fluent-bit
